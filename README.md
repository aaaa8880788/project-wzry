### 项目名称：

王者荣耀手机端官网和管理后台全栈项目

### 项目描述：

该项目主要是运用了Vue2.0开发的仿王者荣耀手机端项目，官网页面主要用的原生css对页面进行布局，管理后台页面主要用了element-ui进行布局编写，接口部分数据库采用的是mongodb，结合node.js知识通过express开发，增删改查接口运用了通用的crud接口方法，有效的优化了代码量，网络请求部分采用的axios方法
